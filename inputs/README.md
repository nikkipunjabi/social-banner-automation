# Speaker Photo

Before running the banner generator, drop the speaker's profile photo in this folder and name it:

```
speaker_photo.jpg   (or speaker_photo.png)
```

## Requirements
- **Format:** JPG or PNG
- **Recommended:** Square or portrait crop, face centred, clean background
- **Minimum size:** 600 × 600 px (higher is better — it scales down, never up)

## How to upload via GitHub web UI
1. Open this repo on GitHub
2. Navigate to the `inputs/` folder
3. Click **Add file → Upload files**
4. Drag your photo in, name it `speaker_photo.jpg`
5. Commit directly to `main`
6. Then go to **Actions → Generate Webinar Banners → Run workflow**
